Assignments


Basic HTML

With our first assignment, we were meant to show our recently developed skills in the most basic of html code. We showed this by creating a simple website based on whatever topic we wanted and for this assignment, I decided to delve into how scary our planet's oceans are compared to space.

Basic CSS

This second assignment now takes us away from the basics of html, but instead to its' counterpart...CSS styles. Here we applied the basic system of making a website look "pretty" and easier on the eye. With the use of classes, divs, and id's, we were able to make not only our 2nd assignment pretty, but also our homepage.

FlexBox

This third assignment was to demonstrate that we as student can somewhat understand how to use flex box since our next assignment is purely flexbox use. While using these skills is definitely difficult, I know mastering it now wil make the future much much easier.

Recreating CSS

This is by far the hardest assignment so far in CSCE242 as we now had to use our recent knowledge of css layout to recreate a page of our universities Nursing department. I had a lot of trouble with this one but I fully understand how to use flex box and properly set up media queries properly.

JavaScript

This assignment was to showcase our understanding of java script and its different uses. Of course this is only a small example but I'm already starting personal projects of my own to utilize different aspects of java. The calendar java aspect is probably the most important out of the ones showcased here and will most definitely be used in my own project.

Toggle Menu

In this assignment we are still working with javascript but now we are going way more indepth with it and having more java code to write. In this assignment we had to code a range slider, date() counter, and a good bit of listening events. The toggle menu function was the star of the show however as we will be using these in our actual websites.
