During this semester in Web Applications class, I will be able to understand and apply the basics to html, Github, and more. In a more formal sense, the student will be able to apply client-server applications. While this class is said to be challenging, it will be benificial in the long run.

Assignments:

Assignment 01 (Basic HTML)

Assignment one delved into how our future project websites will be formed. We went over the very basics and with those basics learned who to create a simple html website.

Assignment 02 (Basic CSS)

Assignment two delved into the basics of using css within an html Website. The basics of learning how to make a website "pretty" went into play and in turn, we made another website to further our knowledge on what is expected of our final website.
